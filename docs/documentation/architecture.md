# Architecture Overview

The Malabi Genesis Registry is designed as a structured identity layer for the Malabi Universe ecosystem.

## Core Components
- Registry data in data/malabi_registry.json
- Explorer front-end in docs/
- AI knowledge layer in ai/prime_oracle_knowledge.json
- Validation tooling in scripts/

## Design Principles
- Professional and scalable structure
- Clear mathematical metadata
- AI-ready identity representation
- Long-term Web3 compatibility
